         If your using Eclipse Color Theme plugin  (http://www.eclipsecolorthemes.org/?view=plugin)
         import the *.xml file.

         See http://www.eclipsecolorthemes.org/?view=how-to-use for a video of how it's
         done if you don't know what to do.
